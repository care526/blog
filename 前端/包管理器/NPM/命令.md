### npm init
初始化当前项目，指定作者，包名等基础信息

### npm install
```
npm i  # 别名
npm i XXX --save  # 安装XXX为线上依赖
npm i XXX --save-dev  # 安装XXX为本地依赖
```

### npm uninstall
```
npm unstall XXX --save  # 移除XXX包依赖
```

### npm update
更新当前包版本

### npm run
会在package.json中去寻找scripts字段，去执行对应的命令
如：npm run serve  /  npm run build
