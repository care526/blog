## if else
```
var ten int = 11
if ten > 10 {
  // XXX
} else if ten < 5 {
  // XXX
} else {
  // XXX
}
```
```
if ten := 10; ten > 10 {}
```
## for
```
for i := 0; i < 10; i++ {  // 初始语句可以不写，但是后面的分号(;)必须写
    sum += i
}
```
```
for {}  // 无限循环，在循环体中使用break来跳出
```
