## 引用
### 全路径｜相对路径
包的绝对路径就是“$GOROOT/src 或 $GOPATH/src”后面包的源码的全路径
```
import "lab/a"
import "../a"
```
### 引用格式
```
import "fmt"  // 标准引用
fmt.Println("xxx")

import F "fmt"  // 别名引用
F.Println("xxx")

import . "fmt"  // 命名空间合并，引用
Println("xxx")  // 直接使用

import _ "fmt"  仅执行包初始化 init 函数
```
