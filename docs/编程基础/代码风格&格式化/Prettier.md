## 配置文件
### 支持的类型
- package.json 中的 prettier 属性
- .prettierrc 文件 格式支持：JSON / YAML
- .prettierrc.json
  .prettierrc.yml
  .prettierrc.yaml
  .prettierrc.json5
- .prettierrc.js
  .prettierrc.cjs
  prettier.config.js
  prettier.config.cjs
  > exports an object using module.exports
- .prettierrc.toml
