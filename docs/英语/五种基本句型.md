# 五种基本句型

主语 + (不及物)动词
无承受者

主语 + (及物)动词 + 宾语
一个承受者

主语 + (双及物)动词 + 间接宾语 + 直接宾语
两个承受者

主语 + (复杂及物)动词 + 宾语 + (宾语)补语
一个承受者

主语 + (系)动词 + (主语)补语<表语>
赋予信息
